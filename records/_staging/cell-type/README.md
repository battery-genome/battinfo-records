# Staging Cell Types

Place shared cell-type records here while they are under editorial review and not yet accepted into the curated corpus.

Use one JSON file per candidate record in this directory.
Use the intended curated record id as the filename when it is already known.
If the record id is still ambiguous, a provisional descriptive filename is acceptable until review assigns the final disambiguator.

When a record is approved, validate and promote it with BattINFO tooling into `records/cell-type/<record-id>/record.json`.
