# Curated Cell Types

This directory holds accepted shared cell-type records.

- Use one directory per record id.
- Keep the main artifact at `record.json`.
- Add metadata in the mirrored `metadata/` tree only when it materially helps editorial review or provenance tracking.
- Do not place unreviewed personal drafts here.

The `_template-cell-type/` directory is an example only. Directories beginning with `_` are not part of the curated corpus.
